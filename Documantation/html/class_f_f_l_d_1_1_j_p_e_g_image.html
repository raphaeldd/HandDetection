Detailed Description

The JPEGImage class allows to load/save an image from/to a jpeg file, and to apply basic operations such as resizing or cropping to it. The pixels are stored contiguously in row-major order (scanline 0: RGB RGB RGB, scanline 1: RGB RGB RGB...)
