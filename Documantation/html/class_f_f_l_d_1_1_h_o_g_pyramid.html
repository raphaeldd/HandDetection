Detailed Description

The HOGPyramid class computes and stores the HOG features extracted from a jpeg image at multiple scales. The scale of the pyramid level of index i is given by the following formula: 2^(1 - i / interval), so that the first scale is at double the resolution of the original image). Each level is padded with zeros horizontally and vertically by a fixed amount. The last feature is special: it takes the value one in the padding and zero otherwise.

Note
Define the PASCAL_HOGPYRAMID_FELZENSZWALB_FEATURES flag during compilation to use Felzenszwalb's original features (slower and not as accurate as they do no angular interpolation, provided for compatibility only).
Define the PASCAL_HOGPYRAMID_DOUBLE to use double scalar values instead of float (slower, uses twice the amount of memory, and the increase in precision is not necessarily useful).
