Detailed Description

The Patchwork class computes full convolutions much faster than the HOGPyramid class.

Patchwork::Patchwork ( const HOGPyramid &  pyramid)

Constructs a patchwork from a pyramid.

Parameters
[in]pyramidThe pyramid of features.
Note
If the pyramid is larger than the last maxRows and maxCols passed to the Init method the Patchwork will be empty.
Assumes that the features of the pyramid levels are zero in the padded regions but for the last feature, which is assumed to be one.
